Step-by-Step

Go live in
one afternoon

No technical experience needed. No credit card. No contracts. Follow these steps and your community hub will be live today.

1
Create a free GitHub account Go to github.com and sign up. Create a repository named yourusername.github.io — this becomes your main free URL automatically.
2
Save & upload this HTML file Save this page as index.html, upload it to your GitHub repo. It goes live in under 2 minutes at yourusername.github.io.
3
Build your community wiki on Notion Create pages in Notion for guides, FAQs, announcements. Set each page to "Share to web" — instantly public, no cost.
4
Set up info pages on Google Sites Use Google Sites for event calendars, resource directories, or contact forms. Zero coding required.
5
Deploy extras on Netlify or Vercel Any extra pages go on Netlify (drag & drop). Any interactive tools or apps go on Vercel (connect to GitHub).
6
Update all links on this main page Replace the placeholder links on this page with your real URLs. Everything now connects from one hub.

Quick Reference

What goes where

  • Main landing page → GitHub Pages
  • Community wiki → Notion
  • Event calendar → Google Sites
  • Resource directory → Google Sites
  • Community blog → Hashnode (free)
  • Interactive tools → Vercel
  • Extra static pages → Netlify
  • Community chat → Discord (free)

Total cost

$0.00

Every platform is free tier. No hidden fees.

Custom domain (optional)

Your free github.io URL works perfectly and is shareable immediately. If you want a custom domain later, Porkbun or Namecheap offer them for ~$10/year.
